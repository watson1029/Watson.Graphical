### 安装
Install-Package Watson.Graphical -Version 1.1.0
### 方法列表
* LineToLine 两条直线关系判断
* LineToPolygon 直线与多边形关系判断
* LineToCircle 直线与圆形关系判断
* PolygonToPolygon 两个多边形关系判断
* PolygonToCircle 多边形与圆形关系判断
* CircleToCircle 两个圆形关系判断
